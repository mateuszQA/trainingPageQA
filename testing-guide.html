Website Icon (Favicon)

What is it? The tiny icon next to your website name in the browser tab.

How to Test:

  1. Look at the browser tab at the top
  2. Is there a custom icon next to the website name?
  3. Bookmark the page (Ctrl+D) and check if icon appears

‚úÖ GOOD:

Custom icon appears in tab and bookmarks

‚ùå BAD:

No icon or generic browser icon - website looks unprofessional

How to Test:

  2. Click the phone/tablet icon (üì±)

  4. Check if you can read text without scrolling sideways

Form Testing

What is it? Testing contact forms, login forms, signup forms to make sure they work.

How to Test:

  1. Fill out the form with correct information first
  2. Try submitting - do you get confirmation?
  3. Leave required fields empty and try to submit
  4. Try wrong email format (like "test@")
  5. Put numbers in name fields
  6. Try very long text in fields
Email Testing Examples:
  • ‚úÖ Good: john@gmail.com
  • ‚ùå Bad: john@ (missing domain)
  • ‚ùå Bad: @gmail.com (missing name)
  • ‚ùå Bad: john.gmail.com (missing @)

üí° Pro Tip:

Always test with wrong information first - this finds most bugs!

Buttons & Links

What is it? Making sure all buttons and links actually work when clicked.

How to Test:

  1. Click every button on the page
  2. Click every link
  3. Does something happen when you click?
  4. Do links go to the right page?
  5. Try hovering over buttons - do they change color?

‚úÖ What Should Happen:

  • Button changes color when you hover
  • Button does something when clicked
  • Links take you to correct pages
  • No broken links (404 errors)

‚ùå Red Flags:

  • Nothing happens when you click
  • Button looks disabled but should work
  • Link goes to wrong page
  • "Page not found" errors

Images & Pictures

What is it? Checking that all images load properly and look good.

How to Test:

  1. Scan the page for missing images
  2. Look for broken image icons (üì∑)
  3. Check if images look stretched or squished
  4. Try on slow internet - do images load?

‚úÖ Good Images:

  • All pictures load completely
  • Images look clear and not pixelated
  • Pictures fit properly in their spaces
  • Images scale nicely on mobile

Pop-ups & Windows

What is it? Testing windows that appear on top of the main page.

How to Test:

  1. Look for X button in corner
  2. Try clicking the X button
  3. Look for Cancel or Close button
  4. Try clicking outside the pop-up
  5. Press Escape key on keyboard

‚ùå CRITICAL PROBLEM:

If you can't close a pop-up, users are trapped! This is a serious bug.

‚úÖ Pop-up Should:

  • Have visible X button
  • Close when X is clicked
  • Close when you click outside it
  • Close when you press Escape

How to Test:

  1. Search for something you know exists
  2. Try common words like "music" or "help"
  3. Search for something that doesn't exist
  4. Try searching with typos
  5. Try searching with nothing (empty search)

Audio & Video

What is it? Testing that music players and videos work properly.

How to Test:

  1. Make sure your volume is turned up
  2. Click the play button
  3. Listen - do you hear sound?
  4. Try the volume slider
  5. Test pause button
  6. Try with headphones

‚ùå Common Audio Problems:

  • Play button doesn't work
  • No sound comes out
  • Volume slider doesn't change volume
  • Music file is missing

üí° Testing Tip:

Always test audio in a quiet room with working speakers or headphones!
